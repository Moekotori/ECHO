常见问题（FAQ）

支持哪些系统？

当前主要提供 Windows 安装包（NSIS）。macOS / Linux 版本尚需在对应平台构建与测试。

如何检查更新？

应用支持 GitHub Releases 的自动更新（electron-updater），也可以在设置中手动检查更新。

下载的安装包如何校验？

下载页会提供 SHA256 校验码，可在 PowerShell 执行 Get-FileHash "ECHO Setup.exe" -Algorithm SHA256 对比。

出现播放/音频问题怎么办？

尝试切换输出设备或关闭独占模式；更新声卡驱动并重启应用，若仍异常请上传日志至仓库 Issues。

如何贡献或反馈 bug？

欢迎在 GitHub 仓库提交 Issue 或 PR： https://github.com/Moekotori/Echoes
