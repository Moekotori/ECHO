# osu! Beatmap Search Plugin

This example plugin adds an `osu!` button beside the lyrics button in ECHO's main player.

When clicked, it opens the official osu! beatmap search page and searches using the current track title and artist.

## Install

1. Open ECHO.
2. Open the plugin manager.
3. Choose `Install`.
4. Select this folder:

`plugin development/osu-beatmap-search`

## Notes

- The button is disabled when no current track is selected.
- Search opens on the official osu! site: `https://osu.ppy.sh/beatmapsets`
